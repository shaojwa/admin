#### windows
```
win+m  // Minimize all windows.
```
#### ppt
```
ctrl+m // Add a new slide.
Ctrl+B // Make selected text bold.
```
#### excel
```
tab    // Move one cell to the right in a worksheet
```
#### mobaxterm
```
F4  // show mobaxterm
```
