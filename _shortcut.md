https://support.microsoft.com/en-us/help/12445/windows-keyboard-shortcuts

#### windows
```
win+m  // Minimize all windows.
```
#### ppt
```
ctrl+m // Add a new slide.
Ctrl+B // Make selected text bold.

alt+h  // 开始选项卡
alt+q  // 视图选项卡
alt+n  // 插入选项卡
```
#### excel
```
tab    // Move one cell to the right in a worksheet
```
#### mobaxterm
```
F4  // show mobaxterm
```
