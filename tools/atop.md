MEM列：
```
tot   ：总的物理内存
cache ：page cache：页缓存。
dirty ：cache中脏的部分，也就是需要下刷到磁盘上的。
buff  ：文件系统元数据占用内存。
slab  ：内存分配机制占用的内存。
```
