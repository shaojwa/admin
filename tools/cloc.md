#### 示例
```
cloc rcache
cloc rcache --by-file           // 列出详细的文件
cloc rcache --counted=cloc.out  // 输出统计到的文件
```
#### 过滤选项

#### 输入选项
```
--extract-with=<cmd>  // 加压抽取的命令，比如针对输入文件是压缩包的情况。
--list-file=<file>    // 指定需要处理的文件
```
#### 处理选项
```
--ignore-whitespace   // 比较时忽略空格
--ignore-case         // 比较时忽略大小写
```
